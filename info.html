Hybrid

Setups where the bed is facing the north/south axis.

Modern

Setups where the bed is facing the east/west axis.

Multiblock

Has beds on both axes. If you choose to do 2 obby instead, one of the two bed positions is the main one and is listed as the setup type.

Multiblock Adjusted

A standing block where you are standing diagonal to the two bed positions made by placing 3 obby.

Report a Problem or Suggest an Improvement

Found incorrect coordinates, missing towers, or have a suggestion? Submit an issue on GitHub and help improve the cheat sheet for everyone!

  • Incorrect Data: Wrong coordinates, Y-levels, or setup types
  • Missing Towers: Towers that should be added to the list
  • New Setups: Alternative setups for existing towers
  • Bug Reports: Issues with the website functionality
  • Feature Requests: Ideas for new features or improvements
Submit an Issue on GitHub

How to Submit a Good Issue

When submitting an issue, please include:

  • A clear, descriptive title
  • The specific tower name (if applicable)
  • The expected vs. actual behavior or data
  • Screenshots if helpful
  • Any additional context that might help
